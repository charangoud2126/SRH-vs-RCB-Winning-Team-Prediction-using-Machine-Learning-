# 🏏 IPL Match Winner Prediction (SRH vs RCB)

##  Overview
This project uses **Machine Learning** to predict the winning team between **Sunrisers Hyderabad (SRH)** and **Royal 

Challengers Bangalore (RCB)** based on match conditions, team stats, and historical data.

The goal is to build a predictive model that can estimate match outcomes with high accuracy using data-driven techniques.

---

##  Features

-  Data preprocessing & feature engineering  

-  Machine Learning model training  

-  Model evaluation (Accuracy, Confusion Matrix)  

-  Match winner prediction system  

-  Easy-to-use prediction input pipeline
